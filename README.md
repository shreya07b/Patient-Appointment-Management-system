
#  Healthcare Management Platform

A modern, full-stack healthcare management system designed to streamline hospital operations, patient care, and administrative workflows through a responsive and intuitive UI.

##  Overview
It provides role-based access for patients, doctors, and administrators to manage appointments, departments, and records in one unified platform.

---

##  Key Features
- Patient management
- Doctor profiles & availability
- Appointment scheduling
- Department management
- Admin dashboard & workflows
- Secure authentication
- Responsive UI

---

## 🏗️ Architecture
Frontend (React + Tailwind) → Backend (Node/Express) → Database

---

##  Structure
- frontend/
- backend/
- admin/

---

##  Tech Stack
React, Vite, Tailwind, Node.js, Express, Clerk Auth

---

##  Setup
```bash
git clone <repo-url>
```

Frontend:
```bash
cd frontend
npm install
npm run dev
```

Backend:
```bash
cd backend
npm install
npm start
```

Admin:
```bash
cd admin
npm install
npm run dev
```
